### <a name="题目描述">题目描述</a>

输入一个年份（大于 1582 的整数 ），判断这一年是否是闰年，如果是输出 1，否则输出 0。

### <a name="输入格式">输入格式</a>

无

### <a name="输出格式">输出格式</a>

无

### <a name="输入输出样例">输入输出样例</a>

#### 输入 #1

```c++
1926
```

#### 输出 #1

```c++
0
```

#### 输入 #2

```c++
1900
```

#### 输出 #2

```c++
0
```

#### 输入 #3

```c++
2000
```

#### 输出 #3

```c++
1
```

#### 输入 #4

```c++
1996
```

#### 输出 #4

```c++
1
```

### <a name="C++代码">C++代码</a>

```c++
#include <iostream>
using namespace std;

int main() {
    int year;
    cin >> year;
    if (year % 100!=0 && year % 4 == 0 || year % 400 == 0)
        cout << 1 << endl;
    else
        cout << 0 << endl;
    return 0;
}
```
