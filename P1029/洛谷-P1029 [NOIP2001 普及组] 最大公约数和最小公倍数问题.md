### <a name="题目描述">题目描述</a>

输入两个正整数$x_0, y_0$，求出满足下列条件的 $P, Q$ 的个数：

1. $P,Q$ 是正整数。
2. 要求 $P, Q$ 以 $x_0$ 为最大公约数，以 $y_0$ 为最小公倍数。

试求：满足条件的所有可能的 $P, Q$ 的个数。

### <a name="输入格式">输入格式</a>

一行两个正整数 $x_0, y_0$。

### <a name="输出格式">输出格式</a>

一行一个数，表示求出满足条件的 $P, Q$ 的个数。

### <a name="输入输出样例">输入输出样例</a>

#### 输入 #1

```c++
3 60
```

#### 输出 #1

```c++
4
```

### <a name="说明/提示">说明/提示</a>

P,Q 有 $4$ 种：

1. $3, 60$。
2. $15, 12$。
3. $12, 15$。
4. $60, 3$。

对于 $100\%$ 的数据，$2 \le x_0, y_0 \le {10}^5$。

**【题目来源】**

NOIP 2001 普及组第二题

### <a name="C++代码">C++代码</a>

```c++
#include <cstdio>
#include <algorithm>
using namespace std;

int main() {
    int x, y, i, j, t, ans=0;
    scanf("%d%d", &x, &y);
    for (i=x; i<=y; ++i) {
        if (x * y % i == 0)
            j = x * y / i;
            t = __gcd(i, j);
            if (t==x && i*j*1.0/t==y)
                ++ans;
        }
    printf("%d\n", ans);
    return 0;
}
```
