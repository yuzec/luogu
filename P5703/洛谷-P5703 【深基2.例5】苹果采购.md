### <a name="题目描述">题目描述</a>

现在需要采购一些苹果，每名同学都可以分到固定数量的苹果，并且已经知道了同学的数量，请问需要采购多少个苹果？

### <a name="输入格式">输入格式</a>

输入两个不超过 $10^9$ 正整数，分别表示每人分到的数量和同学的人数。

### <a name="输出格式">输出格式</a>

一个整数，表示答案。保证输入和答案都在int范围内的非负整数。

### <a name="输入输出样例">输入输出样例</a>

#### 输入 #1

```c++
5 3
```

#### 输出 #1

```c++
15
```

### <a name="C++代码">C++代码</a>

```c++
#include <iostream>
using namespace std;

int main() {
    int n,m;
    cin>>n>>m;
    cout<<n*m<<endl;
    return 0;
}
```
