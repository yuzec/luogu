### <a name="题目描述">题目描述</a>

给出三个整数 $a,b,c(0\le a,b,c \le 100)$，要求把这三位整数从小到大排序。

### <a name="输入格式">输入格式</a>

无

### <a name="输出格式">输出格式</a>

无

### <a name="输入输出样例">输入输出样例</a>

#### 输入 #1

```c++
1 14 5
```

#### 输出 #1

```c++
1 5 14
```

#### 输入 #2

```c++
2 2 2
```

#### 输出 #2

```c++
2 2 2
```

### <a name="C++代码">C++代码</a>

```c++
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int n[3];
    cin >> n[0] >> n[1] >> n[2];
    sort(n, n+3);
    cout << n[0] << ' ' << n[1] << ' ' << n[2] <<endl;
    return 0;
}
```
