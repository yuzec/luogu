### <a name="输入格式">输入格式</a>

只有1行：n

### <a name="输出格式">输出格式</a>

只有1行：Fn，保留两位小数。

### <a name="输入输出样例">输入输出样例</a>

#### 输入 #1

```c++
6
```

#### 输出 #1

```c++
8.00
```

### <a name="说明/提示">说明/提示</a>

所有数据：n<=48

如果不知道怎么做，可以先计算出前几项，然后找规律。

### <a name="C++代码">C++代码</a>

```c++
#include <cstdio>
#include <cmath>
using namespace std;

int main() {
    int n;
    double ans;
    scanf("%d", &n);
    ans = (pow((1+sqrt(5))/2.0, n) - pow((1-sqrt(5))/2.0, n)) / sqrt(5);
    printf("%.2f\n", ans);
    return 0;
}
```
