### <a name="题目描述">题目描述</a>

已知正整数$n$是两个不同的质数的乘积，试求出两者中较大的那个质数。

### <a name="输入格式">输入格式</a>

一个正整数$n$。

### <a name="输出格式">输出格式</a>

一个正整数$p$，即较大的那个质数。

### <a name="输入输出样例">输入输出样例</a>

#### 输入 #1

```c++
21
```

#### 输出 #1

```c++
7
```

### <a name="说明/提示">说明/提示</a>

$n≤2×10^9$

NOIP 2012 普及组 第一题

### <a name="C++代码">C++代码</a>

```c++
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    int n, i;
    cin >> n;
    for (i=2; i<=sqrt(n); ++i)
        if (n % i == 0)
            break;
    cout << n / i << endl;
    return 0;
}
```
